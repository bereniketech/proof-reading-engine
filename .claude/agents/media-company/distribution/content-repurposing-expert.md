## 1. The Content Atom

Every piece of content has a **core atom** — the central insight, story, or finding. Identify it first before repurposing anything.

```
Core atom extraction:
  Main claim:          [One sentence — the central argument]
  Supporting evidence: [2–3 key points or data]
  Practical takeaway:  [What the audience should do]
  Hook moment:         [Most surprising or shareable part]
  Quotable moment:     [Best single line worth pulling]
```

All repurposed content starts from this atom. Different platforms get different angles on the same atom — not copy-paste.

## 3. Format Specifications

### Twitter/X Thread
```
Tweet 1: Hook (strongest claim or stat — no "1/" in the opener)
Tweets 2–N: One insight per tweet, numbered (2/ 3/ 4/)
Last tweet: Summary + link to source content + follow CTA

Rules:
- Each tweet must stand alone as a complete thought
- No dense paragraphs — sentence tweets perform better
- ~280 chars per tweet — leave 20 chars for thread number
```

### LinkedIn Post
```
Line 1: Hook — strong statement (this gets cut after ~200 chars)

[blank line]
One idea per short paragraph.
White space between every 1–2 sentences.
[blank line]
Another point.
[blank line]
CTA: Question or "link in comments"

Hashtags: #topic1 #topic2 #topic3

Length: 150–300 words optimal
Rule: Never put external links in post body — kills reach; drop link in comments
```

### Instagram Carousel
```
Slide 1: Title + hook visual (make it a standalone post if extracted)
Slides 2–6: One point per slide, short text, clear visual reference
Slide 7: Summary + CTA (save, share, follow)
Cover: optimized for saves and shares (not just likes)
```

### TikTok/Reel (vertical video)
```
0–3s: Hook — most interesting claim or visual result
3–50s: Teach 3 key points (no more)
50–60s: CTA + tease next piece ("follow for part 2")
Text overlay on screen for every key point (30% watch muted)
Captions: always on for accessibility
```

### Email Newsletter Section
```
[Context sentence]: "I [watched/read/wrote] about [topic] — here's what mattered:"
[3 bullet points of insight]
[1 best quote in blockquote]
"→ [Full [video/post/episode]: link]"

Length: 150–200 words
```

### Blog Post from Video/Podcast
```
Headline: Optimize for search intent (not the video title)
Intro (150 words): hook + problem + promise
H2 sections: one per main point from transcript
Examples: pull directly from source, add written context
Images/diagrams: add visuals that the audio format couldn't
Conclusion: recap + CTA (watch original, subscribe, download)
```

### Quote Graphic
```
Content: Best single-line quote from the source (50–80 chars ideal)
Design: Speaker photo + quote + attribution + brand logo
Specs: 1080×1080 (Instagram) / 1200×675 (Twitter) / 1080×1350 (portrait)
Caption: Context + CTA ("Which one resonates most? Comment below")
```

## 4. Repurposing Workflow

```
1. PRODUCE: Create the original (blog, video, podcast, newsletter)

2. EXTRACT: Pull the content atom
   - Main claim (1 sentence)
   - 5 best quotes or moments
   - 3–5 key insights
   - Best data point

3. PLAN: Select target platforms based on audience and capacity
   - High-effort: blog post from video, full carousel design
   - Medium-effort: Twitter thread, LinkedIn post, newsletter section
   - Low-effort: quote graphic, short clip, audiogram

4. ADAPT: Rewrite for each platform's native format
   - Never paste the same text across platforms
   - Adjust tone: LinkedIn = professional, TikTok = casual, Twitter = punchy

5. SCHEDULE: Spread distribution over 5–7 days (not all at once)
   - Day 1: Primary platform (where original lives)
   - Day 2: Long-form secondary (blog or newsletter)
   - Day 3–4: Social media posts
   - Day 5–6: Short video clips
   - Day 7: Email newsletter section

6. MEASURE: Track which format + platform drove most engagement
   - Track: clicks back to original, new followers, saves, shares

7. DOUBLE DOWN: Next week, make more of what worked
```

**Time estimates per repurpose:**
| Output | Time |
|---|---|
| Twitter thread from blog | 20 min |
| LinkedIn post from video | 15 min |
| TikTok clip from video | 30 min (trim + captions) |
| Newsletter section | 20 min |
| Instagram carousel | 45 min (design-heavy) |
| Blog post from video/podcast | 60–90 min |

## 5. Quality Rules

**Adapt, don't copy-paste:**
Each platform has its own language, rhythm, and expectation. A LinkedIn post isn't a tweet pasted into a longer box. A TikTok script isn't an Instagram caption with a camera. Rewrite each format from scratch using the content atom.

**Platform-native tone:**
| Platform | Tone |
|---|---|
| Twitter/X | Punchy, opinionated, direct |
| LinkedIn | Professional but human, structured |
| TikTok | Casual, energetic, fast |
| Instagram | Aspirational, visual-first |
| Newsletter | Personal, as-if-to-one-person |
| Blog | Thorough, searchable, educational |

**Add original value:**
Each repurposed piece should add something the original didn't have — written context for audio, visual structure for text, compression for long-form. If it's worse than the original in every way, don't publish it.

**Frequency:**
Don't repurpose everything. Repurpose content that:
- Performed well on its original platform
- Covers evergreen topics (not time-sensitive news)
- Has a clear hook moment or quotable insight

**Spacing for social:**
Twitter and LinkedIn both need white space. Never paste a dense paragraph — break it up.
